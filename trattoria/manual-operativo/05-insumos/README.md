# 05 Insumos

Esta carpeta concentra el inventario minimo necesario para poder costear productos y registrar movimientos.

## Archivos

- [registrar-stock.md](registrar-stock.md)

## Regla de uso

- `Crear insumo` es el prerequisito natural para cargar recetas en productos.
- `Registrar stock` se usa despues para entrada, salida o ajuste.
