# 06 Categorias

Esta carpeta documenta la categoria del menu como entidad reutilizable para productos y, cuando hace falta, para promociones.

## Archivos

- [crear-categoria.md](crear-categoria.md)

## Regla de uso

- Usa una categoria normal para productos comunes.
- Activa `Modo Promociones` solo si la categoria se va a usar para agrupar promociones.
