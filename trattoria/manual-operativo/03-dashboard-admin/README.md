# 03 Dashboard admin

Esta carpeta documenta el panel administrativo como punto de entrada del sistema interno. Reune vistas, reglas de negocio, casos de uso y flujos transversales antes de entrar a los modulos especificos.

## Archivos

- [navegar-panel.md](navegar-panel.md)
- [mapa-de-vistas.md](mapa-de-vistas.md)
- [reglas-de-negocio.md](reglas-de-negocio.md)
- [flujos-y-casos-de-uso.md](flujos-y-casos-de-uso.md)
- [caja-y-brechas.md](caja-y-brechas.md)

## Que cubre esta carpeta

- Acceso y navegacion del panel `ADMIN`.
- Dashboard inicial y su lectura operativa.
- Reportes, usuarios, perfil y configuracion como vistas base del admin.
- Reglas transversales de permisos, metricas, cobros, egresos y configuracion.
- Estado actual de la documentacion de caja que hoy existe en Notion pero no como modulo separado en el repo.

## Que no repite

- Los pasos detallados de `Pedidos`, `Insumos`, `Categorias`, `Productos` y `Promociones` ya viven en sus carpetas propias.
- Esta carpeta los referencia cuando forman parte de un flujo del panel, pero no duplica instrucciones.

## Orden recomendado

1. [Navegar panel](navegar-panel.md)
2. [Mapa de vistas](mapa-de-vistas.md)
3. [Reglas de negocio](reglas-de-negocio.md)
4. [Flujos y casos de uso](flujos-y-casos-de-uso.md)
5. [Caja y brechas](caja-y-brechas.md)

## Regla de uso

- Usa esta carpeta despues de [Iniciar sesion](../01-acceso/iniciar-sesion.md).
- Toma esta seccion como marco funcional del panel admin.
- Luego entra al flujo puntual del modulo que necesites operar.
